# Security Policy

## Supported Versions

This repository is an educational prototype. Security fixes are applied on the
default branch.

## Reporting a Vulnerability

If you discover a security issue, please open a private report by contacting
the repository maintainers first instead of disclosing it publicly in an issue.

When reporting, include:

- a clear description of the issue
- steps to reproduce
- potential impact

We will acknowledge valid reports and work on a fix as quickly as possible.
