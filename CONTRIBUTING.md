# Contributing Guide

Thank you for your interest in contributing to this project.

## Ways to Contribute

- report bugs
- suggest improvements
- improve documentation
- submit code changes via pull request

## Before You Start

1. Open an issue to discuss major changes.
2. Keep changes focused and small.
3. Make sure your changes are clear and documented.

## Local Setup

```bash
pip install -r requirements.txt
python scripts/child_like_agent.py
```

## Pull Request Checklist

- [ ] The change is focused and clearly described.
- [ ] Documentation is updated when needed.
- [ ] The demo script still runs successfully.
- [ ] Commit messages are meaningful.

## Code Style

- Keep the code simple and educational.
- Prefer readable names and short functions.
- Avoid unnecessary dependencies.

## Community

By participating, you agree to follow the project's
[Code of Conduct](./CODE_OF_CONDUCT.md).
